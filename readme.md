    대학생 때부터는 서울로 이사와서 지금까지 살고 있습니다
    3. 대학생활 및 군대<br>
    -숭실대학교를 나왔으며 경영학과를 전공했습니다. 중간에 군대는 공군을 나왔으며 
    4. 졸업 이후<br>
    -대학교 시절 강의를 잘 따라가지 못해 방황하다가 취업도 지지부진하게 되어 결국 공무원 시험 준비를 몇년동안 했지만 결과가 좋지 못하여
    취업상담 등을 받다가 이 학원까지 오게 되어 현재 수강중입니다
